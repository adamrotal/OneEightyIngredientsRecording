Cara menjalankan:

- Simpan di directory tertentu
- Gunakan XAMPP atau apapun yang dapat menjalankan sever 
APACHE agar dapat menjalankan .php
- Pastikan PostgreSQL sudah terinstall, dan pastikan ada user 'postgres'
menggunakan password 'pass'
- Jalankan server APACHE
- Buka browser, ketik localhost dan bukalah directory yang sebelumnya
sudah ditentukan
- Program sudah dapat dijalankan.
